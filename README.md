# NextMemory - Choose Your Next Memory 🌍

A modern, interactive travel trip organization website that helps users discover and book their next adventure based on mood and budget preferences.

## 🎯 About the Project

**NextMemory** is a travel platform designed to make trip planning exciting and personalized. Instead of just selling trips, we design memories that last a lifetime. Users can explore destinations based on their vibe (Mountains, Beaches, Cities, Nature) and budget preferences.

## ✨ Features

### 🏠 **Home Page (index.html)**
- Hero section with destination search
- **Pick Your Vibe** section with mood-based exploration:
  - Mountains (Peace, height, cold air)
  - Beaches (Sun, sand, ocean waves)
  - City Trips (Lights, culture, energy)
  - Nature (Forests, trails, raw beauty)
- Trending trips carousel showing popular bookings
- AI-powered trip suggestion tool based on feelings and budget

### 🔍 **Discover Trips (discover.html)**
- Browse available trip packages
- Filter by mood categories

### 📅 **Booking Page (booking.html)**
- Easy trip booking interface
- View available dates and prices

### 🏨 **Destination Details (destination.html, details.html)**
- In-depth information about destinations
- Itineraries and travel tips

### 👤 **User Authentication**
- Login page (login.html)
- Signup page (signup.html)

### 📞 **Contact Page (contact.html)**
- Get in touch with our support team

### 📁 **Images**
- Local image assets folder for trip photos

## 🛠️ Technology Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with CSS variables and Flexbox
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome** - Icon library for beautiful UI elements
- **Unsplash API** - High-quality background images

## 🎨 Key Features Implementation

### Interactive Mood Selection
Users click on mood cards to filter trips by vibe.

### Smart Suggestion Tool
Two-step process asking:
1. How do you want to feel? (Peaceful, Adventurous, Energetic)
2. What's your budget? (Budget Friendly, Luxury)

### Mobile-Responsive Design
- Hamburger menu for mobile navigation
- Responsive grid layouts
- Touch-friendly interface
